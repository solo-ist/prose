# Take a Break

Provide a brief session summary when the user signals they want to stop working.

## Trigger Phrases

Automatically invoke this skill when the user says:
- "let's take a break"
- "let's stop for now"
- "let's pause"
- "I'm done for now"
- "stopping here"
- Or any similar phrase signaling the end of a coding session

## Output Format

When triggered, output:

1. **Brief summary** (1-2 sentences) of what we were working on, focused on making it easy to resume later
2. **Current date and time** from the system

## Example Output

```
We were fixing the annotation fade-in animation to only trigger on document load, not on mouse-out events.

Tuesday, February 3, 2026 at 12:46 AM EST
```
