### **Description**

\
Prose is a minimal markdown editor for the Mac, built for the agentic era — where markdown is everything, your API keys are yours, and AI works alongside you directly.

**Plain Files Always**\
Your documents are .md files — open them in any editor, back them up anywhere, send them to anyone. No proprietary formats, no cloud sync required, no lock-in. They'll outlast this app.

**Bring Your Own Keys**\
Bring your own Anthropic API key. Prose connects directly to Anthropic's APIs with no markup and no middleman. Your conversations stay local on your machine.

**AI Suggested Comments**\
When AI suggests edits, it's because you asked, and Prose shows them as inline comments so it feels like you're collaborating with a person. Accept or reject each change individually, provide feedback, or batch review through two user friendly UIs. Your document stays clean until you say so.\
\
**AI Annotation Tracking**\
When AI edits are accepted and made, you can keep track of human and AI provenance through an imbedded annotation highlight you can toggle off and on.

**Clean, Minimal Editor**\
Pure markdown editing with TipTap. YAML frontmatter — parsed and preserved, never rendered. Light and dark mode. Autosave. Standard keyboard shortcuts. The words are what matter.

Privacy First\
No accounts, no telemetry, no analytics. API keys stored in macOS Keychain. All data stays on your machine. No subscription required.

Open Source Software\
Prose is MIT licensed — the same codebase is on GitHub. Read every line, fork it, build on it. Prose exists to be useful, not to manufacture dependency.

## App Privacy

**Data Not Collected**

Prose does not collect any data. All API keys are stored in macOS Keychain. No analytics, no telemetry, no crash reporting to third parties.

Note: Prose passes your API key and message content directly to Anthropic when you use AI features. See Anthropic's privacy policy for details.
